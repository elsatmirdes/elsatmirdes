## ⭐ Öne Çıkan Projeler

### 🟠 Real-Time Bitcoin Anomali Tespit Sistemi
**Big Data / Streaming** odaklı bir proje: canlı BTC verisini alıp işleyerek anomali tespiti yapar ve bildirim üretir.  
🔗 Repo: https://github.com/elsatmirdes/Real-Time-Anomaly-Detection  
**Stack:** Spark Structured Streaming, Kafka, MLlib (K-Means), Elasticsearch/Kibana, Docker

### 🟣 Odaklanma Takibi & Raporlama (Mobil)
**React Native + Expo** ile geliştirilmiş odak/alışkanlık takibi ve raporlama uygulaması.  
🔗 Repo: https://github.com/elsatmirdes/Odaklanma_Takibi_ve_Raporlama_Uygulamasi  
**Stack:** TypeScript, Expo, SQLite, grafik/raporlama bileşenleri, Tailwind yaklaşımı (nativewind)

### 🔵 Kuaför / Berber MVC Uygulaması
Rol bazlı kullanıcı yönetimi (admin/user/employee) ve randevu mantığı olan MVC projesi.  
🔗 Repo: https://github.com/elsatmirdes/KuaforApp  
**Stack:** C#, MVC, PostgreSQL, REST API

### 🟢 Türkçe Metinden Duygu Analizi ile Müzik Üretme
Türkçe metni çeviri + duygu analizi ile sınıflandırıp duyguya göre müzik üreten çalışma.  
🔗 Repo: https://github.com/elsatmirdes/ai_music  
**Stack:** Python, NLTK, googletrans, PyQt5, pygame, pyknon

### 🟡 Daily — Şifreli Günlük Uygulaması
Kullanıcı girişi sonrası günlük kayıtlarını **.aes** formatında şifreleyerek saklayan masaüstü uygulaması.  
🔗 Repo: https://github.com/elsatmirdes/Daily  
**Stack:** Python, PyQt5, SQLite, pyAesCrypt

### ⚫ İşletim Sistemleri — Linux Kabuk (Shell) Uygulaması (C)
İS dersi kapsamında Linux kabuk uygulaması (fork üzerinden).  
🔗 Repo: https://github.com/elsatmirdes/os_project_f
